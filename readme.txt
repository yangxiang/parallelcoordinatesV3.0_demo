ParallelCoordinates Program

This program is used for paper "Visualizing Clusters in Parallel Coordinates for Visual Knowledge Discovery", Yang Xiang, David Fuhry, Ruoming Jin, Ye Zhao, Kun Huang, PAKDD'12, pp.505-516. DOI: http://dx.doi.org/10.1007/978-3-642-30217-6_42

This package contains a test dataset in the "data" directory. You may create more datasets in this directory by following the format of the test dataset. The datasets used in our work were extracted from the UC Irvine Machine Learning Repository (http://archive.ics.uci.edu/ml/). 

The following of this README file describes how to run the program. 

To run the program, please take the following steps:
(1) upload the program to the /var/www/html of a Linux machine with apache installed.
(2) visit the index page using a web browser.

Please note:
(1) The program may perform differently under different versions of browsers. It may fail under some versions of browsers.  
(2) The current version of the program has been tested under Microsoft Internet Explore 9.
